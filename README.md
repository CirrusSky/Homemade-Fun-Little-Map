## 安装方法
1. 按 `Win + R` 输入 
	```
	%localappdata%\FPVKamikazeDrone\Saved\SaveGames\MapEditor
	```
2. 拖动或复制到此目录下（如：将“Kunkun Battlefield(微调整)”文件夹复制过去），重启游戏。
3. 点击"任务-本地“即可找到本地地图，也欢迎各位二创！！！
4. 地图谁来上传或者使用都无所谓，如果你是花钱买的那么你就被骗了。
